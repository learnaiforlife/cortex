---
description: "Detect, recommend, and install CLI tools that make AI coding agents faster"
---

Invoke the scaffold skill with: toolbox $ARGUMENTS

Examples:
- /scaffold-toolbox              — detect and recommend missing CLI tools
- /scaffold-toolbox install      — install all recommended tools
- /scaffold-toolbox audit        — check for outdated tools and conflicts
- /scaffold-toolbox configure    — set AI agent env vars (USE_BUILTIN_RIPGREP, etc.)
